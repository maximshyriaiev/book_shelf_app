# Book Shelf App

## Overview
Book Shelf App is a professional full-stack application for managing a book collection. It features a React-based frontend for user interaction and a Node.js/Express backend for API services. The app allows users to view a list of books and add new ones via RESTful API calls. This project is designed as a monolithic repository for educational purposes, complying with GDPR via cookie consent and including comprehensive documentation.

## Author
Maksym Shyriaiev

## Features
- **Backend**: Node.js with Express, in-memory storage (extendable to database). API endpoints: GET /api/books (list books), POST /api/books (add book).
- **Frontend**: React with Axios for real API integration, reusable components (BookCard, Button), and GDPR-compliant cookie consent popup.
- **Documentation**: Swagger for API, JSDoc for code, Storybook for UI components.
- **Compliance**: MIT License, Privacy Policy (GDPR), license report.

## Requirements
- Node.js v20+ (tested on v20.11.0)
- npm v10+
- Web browser (for frontend and docs)
- No additional databases required (in-memory storage used)
